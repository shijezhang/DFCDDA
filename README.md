# DFCDDA
Diffusion-enhanced fine-grained cross semantic fusion for drug-disease association prediction

# Requirements
- python 3.8.1
- pytorch 1.9.1
- dgl 0.9.1
- numpy 1.24.4
- pandas 2.0.3
- cudatoolkit 11.3.1
- scikit-learn 1.3.2

# Dataset
- DrugDiseaseAssociationNumber: The validated drug-disease associations
- DrugProteinAssociationNumber: The validated drug-protein associations
- ProteinDiseaseAssociationNumber: The validated disease-protein associations
- DrugFingerprint, DrugGIP: Different metrics for constructing drug-drug similarity networks
- DiseasePS, DiseaseGIP: Different metrics for constructing disease-disease similarity networks
- Protein_sequence, ProteinGIP_Drug, ProteinGIP_Disease: Different metrics for constructing protein-related similarity networks
